# 🔐 Python Random Password Generator

## Project Overview

Password Generator is a Python command-line application that creates secure random passwords based on user-defined lengths.

The application allows users to:

- Generate Multiple Passwords
- Choose Password Length
- Include Uppercase Letters
- Include Numbers
- Ensure Minimum Password Length

## Features

✔ Generate Multiple Passwords

✔ Custom Password Length

✔ Automatic Minimum Length Validation

✔ Random Uppercase Letters

✔ Random Numbers

✔ Easy-to-Use Command-Line Interface

## Technologies Used

- Python 3
- Random Module
- Functions
- Loops
- Lists
- String Manipulation
